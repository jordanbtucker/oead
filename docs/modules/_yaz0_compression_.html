compress

  • compress(): Transform
    • Defined in yaz0/compression.ts:24

    Compresses a stream via the Yaz0 algorithm.

    Warning: This implementation currently performs no real compression, and in fact results in larger files, but it does produce valid Yaz0 files.

    example
    createReadStream('ActorInfo.product.byml')
  .pipe(compress())
  .pipe(createWriteStream('ActorInfo.product.sbyml'))

    Returns Transform

    A Transform stream that accepts data and outputs compressed data.
